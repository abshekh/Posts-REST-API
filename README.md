# Posts-REST-API
Posts API built with express.

## Routes
```
@route   GET /posts/
@desc    Get all posts

@route   GET /posts/:id
@desc    Get a post by id

@route   POST /posts
@desc    Create a new post

@route   DELETE /posts/:id
@desc    Delete a post by id

@route   PATCH /posts/:id
@desc    Update a post by id
```

## Quick Start

### Install Dependencies

```bash
# Install dependencies
npm install
```

### Run

```bash
# Run the server
npm start
```
